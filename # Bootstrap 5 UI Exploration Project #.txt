# Bootstrap 5 UI Exploration Project

## Project Description
This project is a responsive multi-page website built using Bootstrap 5. 
It demonstrates the effective use of reusable UI components such as navbars, 
hero sections, cards, and forms. The project focuses on clean layout composition, 
responsiveness, and modern UI design while following internship guidelines.

---

## Technologies Used
- HTML5
- Bootstrap 5 (via CDN)
- Git & GitHub

---

## Project Workflow
1. Added Bootstrap 5 using CDN for styling and interactivity
2. Explored Bootstrap official examples and documentation
3. Identified reusable UI components such as navbar, cards, and forms
4. Designed three original pages: Home, About, and Contact
5. Applied consistent styling and ensured responsiveness
6. Deployed the project using GitHub Pages

---

## Project Reflection

### How I Built the Project
I started by studying Bootstrap 5 documentation and example pages to understand 
component structure and layout patterns. I then extracted reusable UI components 
and rebuilt them to create original pages. Bootstrap utility classes were used 
to maintain spacing, alignment, and responsiveness across all pages.

### Tools and Resources Used
- Bootstrap official documentation
- GitHub for version control and hosting
- ChatGPT for guidance and understanding concepts

### Challenges Faced
- Maintaining consistent layout across pages  
- Ensuring responsiveness on different screen sizes  

### How I Solved Them
- Reused common components like navbar and footer  
- Used Bootstrap grid system and utility classes effectively  

### Learning Outcome
This project helped me understand how to remix Bootstrap components creatively, 
improve UI/UX sense, and build responsive web pages following professional standards.
